Документы HTML

Все документы HTML должны начинаться с обьявления типа документа: <!DOCTYPE html>.

Сам HTML документ начинается с тэга<html> и заканчивается тэгом </html>.

Видимая часть документа пишется внутри тэгов <body> и </body>.

Пример HTML:

<!DOCTYPE html>
<html>
<title>HTML Tutorial</title>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>
Try it Yourself

Обьявление <!DOCTYPE html>

Обьявленгие <!DOCTYPE> означает типа документа и помогает браузеру отобразить документ корректно.

Тип документа обьявляется единожды и только в начале файла(перед любым тэгом)

Обьявление <!DOCTYPE> не чувствителен к регистру.
