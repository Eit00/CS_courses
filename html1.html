Что такое HTML?

  • Абревиатура HTML означает Hyper Text Markup Language
  • HTML это стандартный язык разметки для создания веб страниц
  • HTML описывает структуру веб страницы
  • HTML состоит из серии элементов
  • HTML также сообщает браузеру как должны выстраиваться элементы

Простой пример HTML документа

Пример HTML:

<!DOCTYPE html>
<html>
<title>HTML Tutorial</title>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

Что такое элемент HTML?

Для обозначение элементов HTML использует элементы. Элемент пишется как начальный тэг, какое-то значение и конечный тэг

Например, <название тэга> Какой-то текст </название тэга>

Тэги делятся на два вида:
  • Одинарные
  • Парные
Парные тэги следует закрывать, для этого используется знак "/" перед названием тэга. Например, <html></html>, <body></body>
Одиночные тэги не закрываются и представлены в единичном варианте№ Например, <br> или <hr>

Веб Браузеры

Цель браузеров это чтение HTML документов и правильное их отображение

Браузер не отображает сами тэги, а вместо этого использует их как инструкции для отображения документа:

Структура Веб страницы

Ниже показана визуализация структуры веб страницы:
<html>
<head>
<title>Page title</title>
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
</body>
</html>

Примечание: Содеражние тэга <body> будет отображаться на странице, а содеражние <title> будет отображено как название страницы в табе
